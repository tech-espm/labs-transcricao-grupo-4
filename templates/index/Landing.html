Audio Transcription

LAB experimental 2025.2 4 Semestre

API de Transcrição baseada em Whisper com Timestamps e Saída JSON Estruturada.

Arquivo de áudio → Dados Analíticos Prontos para Uso

Transforme qualquer arquivo de áudio em uma estrutura detalhada de palavras e frases com timestamps precisos.

Processamento e Saída de Dados Estruturados

1. Transcrição com Whisper

Utilizamos o modelo Whisper para alta precisão. Qualquer formato de entrada (WAV, MP3, M4A, OGG) é aceito

2. Saída JSON por Palavra (Transcript)

O resultado primário é uma lista de palavras com timestamps de início e fim (start/end), perfeito para sincronização de vídeo ou análise linguística.

{
  "word":"olá",
  "start":0.52,
  "end":0.78
}

3. Agrupamento em Sentenças

Transformamos a lista de palavras em frases legíveis usando um algoritmo determinístico baseado em pausas e em um máximo de palavras. Você obtém a frase completa com seus próprios timestamps.

Construído pelo Grupo 4 do 4 semestre do curso de Ciências de Dados e Negócios da ESPM (2025.2)

API Robusta e Limpa

Funcionamento é relavivamente simples: foi utilizado a linguagem python com as bibliotecas Flask (criação e gerenciamento de rotas), FastAPI (parte de backend de envio de arquivo para transcrição e resposta em JSON) e o modelo de IA Whisper da OpenAI

Tecnologia: **Backend Python (FastAPI), Storage **S3** (em produção).
